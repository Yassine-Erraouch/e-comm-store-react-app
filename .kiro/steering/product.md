# Product Overview

This is an e-commerce shoe store React application that allows users to browse and purchase shoes. The application fetches shoe data from an external API (DummyJSON) and provides a shopping cart functionality.

## Key Features

- Product catalog with men's and women's shoes
- Shopping cart with add/remove functionality
- Product filtering by category, brand, price range, and color
- Stock management and inventory tracking
- Discount pricing display
- Product ratings and reviews
- Responsive design for shoe browsing

## Business Domain

The application focuses on shoe retail with features like:
- Product categorization (men's/women's shoes)
- Brand filtering and display
- Discount percentage calculations
- Stock availability tracking
- Shopping cart management
